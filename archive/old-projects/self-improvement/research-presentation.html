القدرات والإبداعية

أدوات لتوليد التقارير والعروض والمحتوى وتنفيذ الكود

Carbone
~2,500
محرك تقارير — يحول JSON + Template (DOCX/XLSX) إلى تقارير PDF/DOCX/XLSX احترافية. فواتير، عقود، تقارير تلقائية!
درجة الفائدة9/10
✅ Node.js 💰 مجاني ⚡ سهل
Presenton
~2,000
بديل مفتوح لـ Gamma.app — يولد عروض تقديمية احترافية من prompt. يصدر PPTX و PDF. فيه MCP Server مدمج!
درجة الفائدة9/10
✅ Docker 💰 مجاني + API ⚡ سهل
PDFKit
~10,000
مكتبة Node.js لإنشاء PDFs برمجياً — نصوص، صور، جداول، charts. بدون أي dependencies خارجية.
درجة الفائدة8/10
✅ Node.js native 💰 مجاني MIT ⚡ سهل جداً
E2B Sandbox
11,000+
بيئات sandbox معزولة لتنفيذ الكود بأمان. microVM لكل عملية — أأمن من Docker المباشر. Free tier: 100 ساعة/شهر.
درجة الفائدة8/10
✅ Cloud API 💰 Free tier 🔧 متوسط
CrewAI
~25,000
بناء "فرق" من AI agents — كل agent له دور وأدوات ومهام محددة. أبسط وأسرع من AutoGen.
درجة الفائدة8/10
✅ Python 💰 مجاني + API 🔧 متوسط
Slidev
~35,000
يحول Markdown إلى عروض تقديمية جميلة مع code highlighting و animations و themes.
درجة الفائدة7/10
✅ Node.js 💰 مجاني MIT
DeepL API
REST API
أفضل ترجمة آلية — أدق من Google Translate. Free tier: 500,000 حرف/شهر.
درجة الفائدة7/10
✅ REST API 💰 Free tier
Agency Swarm
~4,000
Multi-agent orchestration يحاكي هيكل شركة حقيقي (CEO, VA, Developer). يدعم Claude عبر LiteLLM.
درجة الفائدة7/10
✅ Python 💰 مجاني

التكاملات والبراوزر

أدوات التحكم بالمتصفح والتكامل مع الخدمات الخارجية

Playwright MCP
~10,000
MCP server رسمي من Microsoft — يتحكم بالمتصفح عبر accessibility snapshots. لا يحتاج vision models! الأنسب لنظامنا.
درجة الفائدة10/10
✅ TypeScript 💰 مجاني MIT ⚡ npx فقط
Stagehand
~15,000
AI browser automation بـ TypeScript — يجمع بين الكود والـ AI. Self-healing و auto-caching. يعمل على Playwright!
درجة الفائدة9/10
✅ TypeScript native 💰 مجاني ⚡ npm install
Crawl4AI
~50,000
زاحف ويب ذكي يحول المواقع لـ Markdown نظيف. Deep crawl + crash recovery + prefetch 5-10x أسرع. مثالي للـ RAG.
درجة الفائدة9/10
✅ Docker 💰 مجاني 🔧 Docker setup
browser-use
~55,000
الأشهر في مجال browser automation — يجعل المواقع متاحة لـ AI agents. يدعم Claude, GPT-4, أي LLM.
درجة الفائدة9/10
⚠️ Python 💰 مجاني + Cloud
PipedreamHQ MCP
☁️ Cloud
2,500+ APIs مع 8,000+ أدوات جاهزة عبر MCP واحد! تكامل فوري مع أي خدمة تقريباً.
درجة الفائدة9/10
✅ TypeScript 💰 Free tier ⚡ سهل
Skyvern
~15,000
LLM + Computer Vision لأتمتة المتصفح. يشتغل على مواقع ما شافها قبل. مقاوم لتغييرات الـ layout.
درجة الفائدة7/10
⚠️ Python + Docker 💰 مجاني + Cloud
Claude Computer Use
Anthropic
تحكم Claude بالديسكتوب مباشرة — ماوس + كيبورد + screenshots. يحتاج display server (Xvfb/VNC).
درجة الفائدة8/10
⚠️ يحتاج Xvfb 💰 API فقط
MCP Ecosystem
81,200
منظومة MCP الكاملة — Supabase, Google Drive, Calendar, GitHub, Memory, Sequential Thinking, والمزيد.
درجة الفائدة10/10
✅ TypeScript 💰 مجاني 🔧 حسب الخادم
