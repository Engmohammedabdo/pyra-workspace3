### Architecture Lessons Learned ⚠️
1. **Next.js rewrites() CANNOT proxy WebSocket** — only HTTP. Must use custom server with `http-proxy`
2. **Coolify public apps**: `git_repository` must be full URL (`https://github.com/...`), not path
3. **Coolify Dockerfile location**: relative to `base_directory`, not repo root
4. **Coolify env vars**: no `is_build_time` field — just `key`, `value`, `is_preview`
5. **Coolify labels**: gzip middleware can interfere with WebSocket
6. **Separate frontend/backend apps**: causes cross-origin WS issues — combine into one app
7. **HTTP/2 + WebSocket**: curl uses HTTP/2 for HTTPS by default, WS needs HTTP/1.1 upgrade
