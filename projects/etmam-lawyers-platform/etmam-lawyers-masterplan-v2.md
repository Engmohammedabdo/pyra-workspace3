## 3. نموذج الوساطة — العقد الثلاثي

### 3.1 أمثلة عالمية ناجحة للوساطة

| القطاع | المنصة | النموذج | التشابه مع إتمام |

| **البناء** | الاستشاري الهندسي | عقد ثلاثي (مالك + استشاري + مقاول) | **مطابق** — تشبيه الأستاذ حسين |

| **القانون** | LawAdvisor | ربط عميل بمحامي — عمولة | عالي — أقرب نموذج |

### 3.2 مقارنة: الاستشاري الهندسي vs إتمام كوسيط

| العنصر | الاستشاري الهندسي | إتمام كوسيط قانوني |
|--------|-------------------|-------------------|
| **الأطراف** | مالك + استشاري + مقاول | عميل + إتمام + محامي |
| **دور الوسيط** | يراقب جودة التنفيذ ويحمي حق المالك | يتابع سير القضية ويحمي حق العميل |
| **مصدر الدفع** | المالك يدفع للاستشاري | العميل يدفع لإتمام رسوم خدمة |
| **القيمة المضافة** | خبرة فنية + رقابة | خبرة قانونية + متابعة + تفاوض |
| **مدة العلاقة** | طول مدة المشروع | طول مدة القضية |
| **المسؤولية** | مسؤولية مهنية عن الإشراف | مسؤولية عن جودة التوصية والمتابعة |

### 3.3 نماذج التسعير الممكنة

| النموذج | الوصف | المميزات | العيوب |
|---------|-------|----------|--------|
| **رسوم ثابتة على العميل** | 500-1,500 درهم رسوم إحالة ومتابعة | بسيط وواضح | قد يردع العملاء |
| **نسبة من أتعاب المحامي** | 10-15% من قيمة الأتعاب | إيراد أعلى مع القضايا الكبيرة | يحتاج شفافية في الأتعاب |
| **اشتراك المحامي + رسوم العميل** | محامي يدفع 500-1,500/شهر + عميل يدفع 750 | إيراد مزدوج ومستقر | أعقد في الإدارة |
| **عمولة نجاح (Success Fee)** | إتمام تاخذ نسبة بعد كسب القضية | جذاب للعميل | إيراد غير مضمون |

## 4. القيود القانونية — الإمارات

### 4.1 إعلان المحامين
- قانون تنظيم مهنة المحاماة في الإمارات يضع قيود على **الإعلان المباشر** للمحامين
- المحامي لا يقدر يعمل إعلانات مبالغ فيها أو يوعد بنتائج
- **لكن:** لا يوجد قانون صريح يمنع **المنصات الوسيطة** من الإعلان عن خدمات ربط العملاء بالمحامين

### 4.2 الوساطة القانونية
- لا يوجد تنظيم صريح لمنصات الوساطة القانونية الرقمية في الإمارات
- **الفرصة:** إتمام بحكم ارتباطها بمحاكم الشارقة تقدر تحصل على تغطية قانونية أقوى من أي كيان خارجي
- **التوصية:** استشارة قانونية رسمية قبل الإطلاق لتحديد الإطار التنظيمي

### 4.3 حماية المستهلك
- قانون حماية المستهلك الإماراتي ينطبق على الخدمات
- **العقد الثلاثي** يوفر حماية إضافية لكل الأطراف — نقطة قوة

## 2. نموذج الوسيط المفاوض — العقد الثلاثي

### 2.1 كيف يعمل بالتفصيل

```
┌─────────────┐     طلب خدمة     ┌──────────────┐     تنفيذ      ┌─────────────┐
│             │ ──────────────► │              │ ─────────────► │             │
│   العميل    │                 │  إتمام       │                │   المحامي   │
│             │ ◄────────────── │  (الوسيط)    │ ◄───────────── │             │
└─────────────┘   متابعة/تقارير └──────────────┘   تقارير أداء   └─────────────┘
       │                              │                               │
       └──────────── عقد ثلاثي ────────┴───────────────────────────────┘
```

**مسار الخدمة:**

| الخطوة | الإجراء | المسؤول |
|--------|---------|---------|
| 1 | العميل يُنهي إجراءاته في إتمام ويحتاج محامي | العميل |
| 2 | موظف إتمام يقيّم الحاجة ويحدد التخصص المطلوب | إتمام |
| 3 | عرض المحامين المتاحين (بروفايلات + فيديوهات) | إتمام |
| 4 | العميل يختار محامي (أو إتمام يرشح الأنسب) | العميل + إتمام |
| 5 | إتمام يتفاوض على الأتعاب نيابة عن العميل | إتمام |
| 6 | توقيع عقد ثلاثي (عميل + إتمام + محامي) | الثلاثة |
| 7 | المحامي يبدأ العمل — إتمام يتابع المراحل | المحامي + إتمام |
| 8 | تقارير دورية للعميل عن سير القضية | إتمام |
| 9 | تقييم الخدمة بعد الانتهاء | العميل |

### 2.2 المقارنة مع الاستشاري الهندسي

| المعيار | الاستشاري الهندسي | إتمام كوسيط قانوني |
|---------|-------------------|---------------------|
| **الدور** | يصمم ويشرف على المقاول | يختار ويتابع المحامي |
| **العلاقة بالمنفذ** | مستقل تماماً عن المقاول | مستقل عن المحامي |
| **التفاوض** | يحدد المواصفات والتكلفة | يفاوض الأتعاب ونطاق العمل |
| **المتابعة** | زيارات ميدانية + تقارير | متابعة المراحل القانونية + تقارير |
| **ضمان الجودة** | شهادات إتمام مرحلية | تقييم أداء المحامي |
| **فض النزاعات** | يحكم بين المالك والمقاول | يتوسط بين العميل والمحامي |
| **المسؤولية** | مسؤولية إشرافية (لا تنفيذية) | مسؤولية وساطة (لا قانونية) |

**الفارق الجوهري:** الاستشاري الهندسي لديه صلاحية فنية (يقبل أو يرفض أعمال)، بينما إتمام وسيط إداري وتفاوضي — لا يتدخل في الرأي القانوني للمحامي.

### 2.3 مسؤوليات كل طرف

| الطرف | المسؤوليات | ما لا يتحمله |
|-------|-----------|--------------|
| **العميل** | • دفع الرسوم في مواعيدها • تقديم المستندات المطلوبة • الحضور عند الحاجة • الالتزام بقرارات المحامي المهنية | • أتعاب إضافية غير مذكورة بالعقد • خسارة القضية (ليس ضماناً) |
| **إتمام (الوسيط)** | • اختيار محامين مؤهلين ومرخصين • التفاوض على أتعاب عادلة • متابعة سير القضية • تقديم تقارير للعميل • التدخل في حال تأخر أو إهمال | • المسؤولية عن نتيجة القضية • تقديم استشارة قانونية • استبدال المحامي بدون مبرر |
| **المحامي** | • تقديم الخدمة القانونية بأعلى جودة • الالتزام بالجدول الزمني • تقديم تقارير لإتمام • الشفافية في التطورات • الحضور والترافع | • أتعاب أقل من المتفق عليه • العمل بدون أتعاب مقدمة • ضمان نتيجة القضية |

## 4. استراتيجية التعامل مع المحامين — "تجنب العداوات"

هذه نقطة حرجة ذكرها الأستاذ حسين. إتمام يجب أن يظهر **داعماً** للمحامين لا منافساً لهم.

### المبادئ:

| المبدأ | التطبيق |
|--------|---------|
| **إتمام يجلب لهم العملاء** | المحامي يحصل على عملاء جاهزين بدون تسويق |
| **الفيديوهات ترفع من بروفايلهم** | إتمام يصوّر محتوى احترافي مجاناً للمحامي |
| **لا حصرية** | المحامي حر يقبل عملاء من خارج إتمام |
| **تقييمات إيجابية** | نظام التقييم يُبرز النقاط الإيجابية أولاً |
| **شراكة لا تبعية** | المحامي شريك مستقل وليس موظف |
| **رسوم تنافسية** | عمولة إتمام أقل من تكلفة التسويق الذاتي للمحامي |

### حزمة المحامي الشريك:

| الميزة | القيمة |
|--------|--------|
| فيديو تعريفي احترافي | تصوير + مونتاج مجاناً (قيمتها ~3,000 درهم) |
| بروفايل على المنصة | ظهور دائم + SEO |
| إحالة عملاء مؤهلين | 3-10 عملاء/شهر |
| متابعة إدارية | إتمام يتابع الدفعات والمواعيد |
| تقارير أداء | إحصائيات شهرية |

## 3. تحليل نقطة التعادل (Break-even)

### 3.1 المعادلة الأساسية

```
نقطة التعادل = التكاليف الثابتة الشهرية ÷ الإيراد لكل قضية

حيث:
- التكاليف الثابتة = رواتب + تشغيل + إيجار
- التكاليف المتغيرة = تسويق (مرتبط بعدد العملاء)
```

### 3.2 Break-even لكل سيناريو

#### السيناريو 1 (رسوم على العميل فقط)

| المؤشر | القيمة |
|--------|--------|
| إيراد/قضية | 1,000 درهم |
| تكاليف ثابتة شهرية (مرحلة 0) | 12,500 درهم |
| **قضايا مطلوبة/شهر للتعادل** | **13 قضية** |
| 3 محامين × 3 قضايا = 9 قضايا/شهر | ❌ **لا يكفي!** |
| **متى يتحقق التعادل؟** | **شهر 8-10** (بعد زيادة عدد المحامين لـ 5+) |
| **كم محامي نحتاج؟** | **5 محامين (× 3 قضايا = 15 قضية)** |

#### السيناريو 2 (رسوم على المحامي فقط)

| المؤشر | القيمة |
|--------|--------|
| إيراد/محامي/شهر | 1,500 اشتراك + 1,500 عمولات = 3,000 درهم |
| تكاليف ثابتة شهرية | 12,500 درهم |
| **محامين مطلوبين للتعادل** | **5 محامين** |
| البداية مع 3 محامين | ❌ **لا يكفي!** |
| **متى يتحقق التعادل؟** | **شهر 7-9** (بعد إضافة محامي 4 و 5) |

#### السيناريو 3 (رسوم على الطرفين) ⭐

| تكاليف ثابتة شهرية (مرحلة 0) | 12,500 درهم |
